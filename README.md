# 🏥 Medical Sales App

A professional, enterprise-grade Pharmacy Management System and Medical Sales Dashboard. This application provides a streamlined interface for managing pharmacy operations, sales tracking, and user roles with high-performance metrics and modern aesthetics.

![Medical Sales App](frontend/src/assets/hero.png)

## ✨ Features

- **🛡️ Secure Authentication**: Robust login system with role-based access control (RBAC).
- **📊 Admin Dashboard**: Comprehensive overview of sales, inventory, and user performance.
- **🖥️ Operator Interface**: Simplified workflow for sales representatives and pharmacy staff.
- **⚡ High Performance**: Built with FastAPI and React for near-instant response times.
- **🎨 Premium UI/UX**: Sleek dark mode, glassmorphism components, and smooth animations.
- **📈 Real-time Analytics**: Dynamic charts and metrics to track business growth.

## 🚀 Tech Stack

### Frontend
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Premium Custom Design)
- **Deployment**: [Netlify](https://www.netlify.com/)

### Backend
- **Language**: [Python 3.10+](https://www.python.org/)
- **Framework**: [FastAPI](https://fastapi.tiangolo.com/)
- **Database**: SQLite (SQLAlchemy ORM)
- **Security**: JWT Authentication & Bcrypt hashing
- **Deployment**: [Render](https://render.com/)

## 🛠️ Installation & Setup

### Prerequisites
- Python 3.10+
- Node.js & npm

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # Windows:
   .\venv\Scripts\activate
   # Unix/macOS:
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the server:
   ```bash
   python main.py
   ```

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
